# Streamlitアプリケーション - マテリアルデータベース

## 概要

このStreamlitアプリケーションは、マテリアルデータベースを簡単に使用できるWebインターフェースを提供します。

## ローカルでの実行

```bash
# 依存パッケージのインストール
pip install -r requirements.txt

# Streamlitアプリの起動
streamlit run app.py
```

ブラウザで http://localhost:8501 にアクセスします。

## 機能

### 🏠 ホーム
- システムの概要
- 最近登録された材料の表示

### 📦 材料一覧
- 登録された全材料の一覧表示
- カテゴリと材料名でのフィルタリング
- 各材料の詳細情報表示

### ➕ 材料登録
- 新しい材料の登録フォーム
- 物性データの動的追加
- カテゴリ選択

### 🔍 検索
- キーワード検索機能
- 材料名、カテゴリ、説明での検索

### 📄 素材カード
- 選択した材料の素材カード表示
- QRコード生成
- 印刷対応
- HTMLダウンロード機能

## データベース

アプリケーションは初回起動時に自動的にSQLiteデータベース（`materials.db`）を作成します。

サンプルデータを追加するには：

```bash
python init_sample_data.py
```

## デプロイ

オンラインで動かすには、`DEPLOY.md`を参照してください。

### Streamlit Cloud（推奨）

1. GitHubにリポジトリをプッシュ
2. https://streamlit.io/cloud にアクセス
3. リポジトリを選択してデプロイ

詳細は `DEPLOY.md` を参照してください。

## トラブルシューティング

### データベースエラー

データベースファイルを削除して再初期化：

```bash
rm materials.db
streamlit run app.py
```

### ポートエラー

別のポートで起動：

```bash
streamlit run app.py --server.port 8502
```
